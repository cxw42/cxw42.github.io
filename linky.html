Here's the link: linky ← right-click or do whatever you want.

Why?

Because sometimes you want to save the target of a URL without opening it in the browser first. Pasting the URL into the address bar doesn't create an actual link, which is what you need. (See, e.g., this request to add such a feature to Firefox.) So here you go!
